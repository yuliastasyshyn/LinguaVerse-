# LinguaVerse AI

LinguaVerse AI — це вебплатформа для спільного вивчення іноземних мов із використанням AI-помічника.

## Основні можливості

- реєстрація та авторизація користувачів;
- навчальні уроки з іноземних мов;
- AI-помічник для пояснень, перекладу та мовної практики;
- модуль перевірки письма;
- тренування вимови;
- словник користувача;
- кімнати для спілкування;
- відеочат;
- сторінка прогресу;
- система завдань;
- адміністративна панель.

## Технології

Frontend:
- React;
- Vite;
- React Router;
- CSS.

Backend:
- Node.js;
- Express;
- PostgreSQL;
- JWT;
- Socket.IO;
- WebRTC;
- Google Gemini API.
