#  LinguaVerse AI

> **LinguaVerse AI** — сучасна вебплатформа для інтерактивного та спільного вивчення іноземних мов із використанням штучного інтелекту.

Платформа об'єднує навчальні уроки, ШІ-помічника, перевірку письма, тренування вимови, словник користувача, чат-кімнати, відеоспілкування та систему відстеження прогресу в одному вебзастосунку.

---

##  Про проєкт

LinguaVerse AI створено як навчальну платформу для ефективного вивчення іноземних мов.

Основна ідея проєкту — поєднати:

*  навчальні матеріали
*  штучний інтелект
*  комунікацію між користувачами
*  інструменти мотивації
*  систему контролю прогресу

## Архітектура системи

Проєкт реалізований за принципом **клієнт-серверної архітектури**.

```text
Frontend (React)
        │
        ▼
Backend (Node.js + Express)
        │
        ▼
PostgreSQL Database
        │
        ├── Authentication (JWT)
        ├── AI Services (Gemini)
        ├── Socket.IO
        └── WebRTC
```

### Frontend

Відповідає за:

* інтерфейс користувача;
* взаємодію з системою;
* роботу ШІ-чату;
* відображення уроків;
* відеозв'язок.

### Backend

Відповідає за:

* REST API;
* бізнес-логіку;
* авторизацію;
* інтеграцію з ШІ;
* роботу з базою даних;
* обмін повідомленнями в реальному часі.

### Database

Зберігає:

* користувачів;
* уроки;
* словник;
* повідомлення;
* прогрес;
* виклики;
* мовні кімнати.

##  AI-модулі

### AI Assistant

Допомагає користувачам:

* пояснювати граматику;
* перекладати тексти;
* генерувати приклади;
* практикувати спілкування;
* вивчати лексику.

### Writing Checker

Функціонал:

* аналіз граматики;
* пошук помилок;
* оцінювання тексту;
* рекомендації щодо покращення письма.

### Pronunciation Trainer

Функціонал:

* перевірка вимови;
* оцінювання схожості фраз;
* автоматичний зворотний зв'язок;
* рекомендації щодо покращення вимови.

---

##  Комунікаційні модулі

### Language Rooms

Користувачі можуть:

* створювати мовні кімнати;
* приєднуватися до існуючих;
* практикувати спілкування.

### Real-Time Chat

Реалізований за допомогою Socket.IO:

* миттєвий обмін повідомленнями;
* список активних користувачів;
* синхронізація повідомлень.

### Video Chat

Реалізований через WebRTC:

* відеозв'язок між користувачами;
* голосове спілкування;
* практика розмовної мови.
