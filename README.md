LinguaVerse AI

LinguaVerse AI — це вебплатформа для інтерактивного та спільного вивчення іноземних мов із використанням AI-помічника. Проєкт поєднує навчальні уроки, персональний словник, перевірку письма, тренування вимови, чат-кімнати, відеоспілкування та систему відстеження прогресу користувача.

Про проєкт
LinguaVerse AI створено як навчальну вебплатформу, яка допомагає користувачам вивчати іноземні мови у зручному інтерактивному форматі. Платформа підтримує самостійне навчання, практику з ШІ-помічником, спілкування з іншими користувачами та контроль власного прогресу.

Основна ідея проєкту — об’єднати в одному вебзастосунку навчальний контент, штучний інтелект, комунікаційні можливості та мотиваційні інструменти.

Архітектура проєкту
Проєкт реалізовано на основі клієнт-серверної архітектури.

Клієнтська частина відповідає за інтерфейс користувача та взаємодію з платформою.
Серверна частина обробляє запити, працює з базою даних, автентифікацією, ШІ-сервісами та комунікаційними модулями.
База даних використовується для збереження користувачів, уроків, прогресу, повідомлень, словника та іншого навчального контенту.

Функціональні модулі
ШІ-помічник
ШІ-помічник використовується для пояснення мовного матеріалу, перекладу, генерації прикладів та допомоги користувачу під час навчання.

Навчальні уроки
Користувач може переглядати уроки, вивчати нову лексику та працювати з навчальним матеріалом.

Перевірка письма
Модуль письма дозволяє користувачу вводити текст і отримувати рекомендації щодо покращення граматики та стилю.

Тренування вимови
Модуль вимови дає можливість користувачу практикувати правильне промовляння слів і фраз.

Персональний словник
Користувач може зберігати слова, переклади та використовувати словник для повторення вивченого матеріалу.

Чат-кімнати
Платформа підтримує кімнати для спілкування, у яких користувачі можуть практикувати мову разом.

Відеоспілкування
Для живої мовної практики реалізовано відеочат між користувачами.

Прогрес користувача
Система прогресу дозволяє відстежувати активність користувача, кількість вивчених слів, виконані завдання та навчальні досягнення.

Адміністративна панель
Адміністратор може керувати користувачами, навчальним контентом, відгуками та іншими даними платформи.

Запуск проєкту
Встановлення залежностей
Для клієнтської частини:

cd frontend
npm install
Для серверної частини:

cd backend
npm install
Запуск серверної частини
npm run backend
Запуск клієнтської частини
npm run dev
Мета проєкту
Метою проєкту є створення сучасної вебплатформи для вивчення іноземних мов, яка поєднує навчальні матеріали, можливості штучного інтелекту, комунікацію між користувачами та персональне відстеження прогресу.

Результат
У результаті розроблено вебплатформу LinguaVerse AI, яка надає користувачам можливість вивчати іноземні мови, практикувати письмо та вимову, спілкуватися з іншими користувачами, користуватися ШІ-помічником і відстежувати власний навчальний прогрес.
